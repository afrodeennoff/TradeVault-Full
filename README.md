## Zero-Error Quick Start

```bash
git clone https://github.com/afrodeennoff/TradeVault-Full.git
cd TradeVault-Full
npm install

# Optional: Initialize database (for future full persistence)
# npx prisma generate
# npx prisma db push

npm run dev
```

Open http://localhost:3000

Everything works out of the box with sample data and localStorage.
